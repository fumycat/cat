# Что это
Консольный клиент для вк

# Фичи

- Список диалогов
- История сообщений (с вложениями и пересланными сообщениями)

# Установка
1. `git clone https://github.com/fumycat/cat.git`
2. `nano .bashrc`

```bash
export API_VERSION='5.60'
export VK_TOKEN='ТОКЕН'
```
3. `source ~/.bashrc`

# Запуск
